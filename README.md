# FMCG-end-end-to-data-engineering-project-using-Databricks
This is an end end to data engineering project using databricks. In FMCG domain, we are building an ETL pipeline for a real life industry use case where a big retail company acquires a small one and we will build the data pipeline to consolidate data from both the companies in a single lakehouse architecture. The tech stack used here is: Python, SQL, Amazon S3, Medallion architecture, Spark, BI dashboard and Genie.
